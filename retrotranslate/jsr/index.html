补丁使用方法

Patching Instructions

你需要准备：

  1. 一份纯净的 [Jet Set Radio (USA) PS2] 游戏镜像文件 (ISO)。请确保你的镜像是完好的，且与补丁要求的版本一致。
  2. 下载本页面提供的汉化补丁文件 (.zip)。
  3. 解压补丁文件，你会得到一个补丁文件 (例如 `.xdelta`) 和一个打补丁工具 (例如 `xdelta UI`)，或者你需要自行下载打补丁工具。
  4. 运行打补丁工具，选择原始游戏镜像文件和汉化补丁文件，然后指定输出路径生成新的、已汉化的游戏镜像。
  5. **重要：** 请务必备份你的原始游戏镜像，以防打补丁过程中出现意外。

打好补丁的镜像文件即可在模拟器或 [若适用] 实机上运行。
