Info (ファイル情報表示)

目的

各種ファイルの属性情報を表示します。

メインウィンドウ

基本操作

  • 入力ファイル (File): 情報を表示するファイルを指定します。
  • 参照 (Browse): 入力ファイルを選択するダイアログを表示します。
  • ファイル種別 (File Type): 情報を表示するファイルの種別を選択します。
  • 表示 (Show): ファイルの情報を表示させます。

Tip

このウィンドウにファイルをドラッグ＆ドロップすると、ファイル情報が表示されます。

ファイル情報表示

表示される情報

pss ファイル (MPEG2 プログラムストリーム)

  • 総パック数
  • 総パケット数
  • プログラムストリーム転送レート (kbit/s, byte/s)
  • 各エレメンタリストリームの情報
    • ストリーム種別、ストリーム ID
    • PES パケット数

ipu 動画ストリーム ファイル

  • 画像サイズ (pixel)
  • フレーム数

spu ストリーム ファイル

  • データフォーマット (PCM Big Endian/Little Endian, ADPCM)
  • 量子化ビット数
  • サンプリング周波数
  • チャンネル数
  • 転送レート (kbit/s, byte/s)
  • サンプル数、再生時間

MPEG ビデオエレメンタリストリーム

  • 画像サイズ (pixel)
  • フレームレート (f/s)
  • 転送レート (kbit/s, byte/s)
  • プロファイルレベル
  • フレーム数、再生時間
  • 各ピクチャタイプの数と割合 (I/P/B/D)
  • PS2 再生互換性
  • IPU 変換互換性

Windows WAVE / Macintosh AIFF オーディオファイル

  • データフォーマット (PCM, etc...)
  • 量子化ビット数
  • サンプリング周波数
  • チャンネル数
  • 転送レート (kbit/s, byte/s)
  • サンプル数、再生時間

操作ボタン

  • 保存 (Save): ファイル情報テキストをファイルに保存します。
  • コピー (Copy): ファイル情報テキストをクリップボードにコピーします。
  • 閉じる (Close): ウィンドウを閉じます。

Tip

ウィンドウのサイズを変更することができます。

コマンドライン操作

GUI の他に、コマンドラインからも同様の機能を利用できます。

基本書式

info [options] target_file

省略形: i [options] target_file

オプション一覧

オプション 説明
-v 作業の進行状況を詳しくレポートします。
-pss pss ファイル (MPEG2 プログラムストリーム) の情報を出力します。
-ipu ipu 動画ストリーム ファイルの情報を出力します。
-ads spu ストリームファイルの情報を出力します。
-video MPEG ビデオエレメンタリストリーム ファイルの情報を出力します。
-wave Windows WAVE オーディオファイルの情報を出力します。
-aiff Macintosh AIFF オーディオファイルの情報を出力します。

使用例

基本的な使用方法

info -pss video.pss

PSS ファイルの情報を表示します。

詳細情報の表示

info -v -video input.m2v

MPEG ビデオファイルの情報を詳細に表示します。

コマンドライン Tips

  • ファイルタイプが明確な場合は、対応するオプションを指定すると処理が速くなります。
  • バッチ処理やスクリプトでの自動化に便利です。
  • -v オプションを使用すると、処理の詳細な進行状況を確認できます。
