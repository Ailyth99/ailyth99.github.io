ps2str ストリームコンバータ ツール

目的

各種メディアファイルを、PlayStation 2 で使用されるファイルへ変換します。

機能

  1. MPEG 動画から IPU 動画への変換
  2. 非圧縮音声データの ADPCM 圧縮およびフォーマット変換
  3. 動画、音声、データ等の複数エレメンタリストリームの多重化
  4. 多重化されているストリームの非多重化
  5. 入力されたデータの属性情報の表示

コマンドライン 書式

一般形式

ps2str [command] [options] [target file]

command 部

処理したい機能を指定します。省略形を使用することもできます。

options 部

各機能によるオプション情報を記述します。

指定可能な内容は、各機能によって異なります。

target file 部

処理対象となるファイルを指定します。

キーワードは、大文字小文字を区別して識別されます。

command 部は、コマンドライン引数の先頭に記述してください。

例：

ps2str demux -video test_stream.pss

command

以下の機能が使用できます。

コマンド名 省略形 機能
convert c MPEG 動画から IPU 動画への変換
encode e 非圧縮音声データの ADPCM 圧縮およびフォーマット変換
mux m 動画、音声、データ等の複数エレメンタリストリームの多重化
demux d 多重化されているストリームの非多重化
info i 入力されたデータの属性情報の表示
help h ヘルプ

GUI 版

コマンドライン引数なして、ps2str を実行するか、ps2strw.exe を直接起動すると、GUI により作業を行うことができます。

(Windows 版のみの機能)
