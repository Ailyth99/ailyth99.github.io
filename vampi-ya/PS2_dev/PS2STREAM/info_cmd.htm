info コマンド

目的

　各種ファイルの属性情報を表示します。

　

書式

command 部

　info (省略形：i)

　

options 部

オプション 意味
-v 作業の進行状況を詳しくレポートします。
-pss pss ファイル (MPEG2 プログラムストリーム) の情報を出力します。
-ipu ipu 動画ストリーム ファイルの情報を出力します。
-ads spu ストリームファイルの情報を出力します。
-video MPEG ビデオエレメンタリストリーム ファイルの情報を出力します。
-wave Windows WAVE オーディオファイルの情報を出力します。
-aiff Macintosh AIFF オーディオファイルの情報を出力します。

　

target file 部

パラメータ１ 対象ファイル

　

出力される情報

pss ファイル (MPEG2 プログラムストリーム)

総パック数

総パケット数

プログラムストリーム転送レート (kbit/s, byte/s)

すべてのエレメンタリストリームについて...

　ストリーム種別、ストリーム ID

　PES パケット数
