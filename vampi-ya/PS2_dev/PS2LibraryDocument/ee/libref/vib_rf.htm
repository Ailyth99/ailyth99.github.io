関数

sceVibGetProfile
振動プロファイルを取得
形式

int scVibGetProfile ( 
	int socket_number,
	unsigned char* profile
)
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能
マルチスレッドセーフでない（割り込み許可状態で呼ぶこと）

引数

 socket_number   ソケット番号 
 profile   取得するプロファイルの先頭アドレス 

返り値

 値   結果 
 0以上   取得したプロファイルサイズ 
 負数   処理失敗 


解説

ソケットにリンクしているコントローラの振動プロファイルを取得します。
現在の振動プロファイルは以下のようになっており、コントローラに存在するアクチュエータのビットが1になり、存在しないときは0になります。

 Byte   Bit   Feature   Size(bit) 
 0   0   小モータ   1 
 1   大モータ   8 
 2   （以下未定）     
 3         
 4         
 5         
 6         
 7         


sceVibSetActParam
アクチュエータへのパラメータ設定
形式

int sceVibSetActParam (
	int socket_number,
	int profile_size,
	unsigned char* profile,
	int data_size,
	unsigned char* data
)
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能
マルチスレッドセーフでない（割り込み許可状態で呼ぶこと）

引数

 socket_number   ソケット番号 
 profile_size   送信するプロファイルサイズ 
 profile   送信するプロファイルの先頭アドレス 
 data_size   送信データのサイズ 
 data   送信データの先頭アドレス 

返り値

 値   結果 
 １   設定成功 
 １以外   設定失敗 


解説

アクチュエータへのパラメータを設定します。
送信するプロファイルに変更したいアクチュエータのビットをそれぞれ1にし、そのアクチュエータのパラメータを送信データに前詰にして格納します。データ送信後はデバイスドライバが送信プロファイルと送信データから各アクチュエータのバラメータを設定します。

(送信データ例) 大モータに値を設定する場合

Profile Data

 Byte   0 
 Bit   7 〜 0 
 Bit Pattern   00000010 

Vibration Data



(送信データ例2) 大モータと小モータを同時に値を設定する場合

Profile Data

 Byte   0 
 Bit   7 〜 0 
 Bit Pattern   00000011 

Vibration Data
